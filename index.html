There are two styles presented here

Both styles include micro-animation on load and hover animation to show label & value. You can switch dark/light mode on individual charts with the toggle. The "print" toggle is meant to show what a printable version is like, which mainly puts values in the legend.

Flat

This flat update is mainly a color palette and some minor features like round bars in bar charts.

Warm Flat

This is a toned-down version of Liquid Glass to improve readability. It uses gradients and shadows that reflect a light source from the top. The effect is more pronounced in dark mode. It's much closer to flat, and Claude described it as "warm flat."
